Overview

Our project utilizes ESP32CAM and ESP32-S3-DevKitC-1 microcontrollers, integrating Cloud Server capabilities for remote monitoring and control. Key components include ESP32CAM, ESP32-S3-DevKitC-1, and SG90 servos, forming a robust surveillance system.

Objective

The primary goal is to create a remote surveillance system that allows users to monitor and control the ESP32CAM's angle remotely. This enhances security measures and ensures real-time response to potential threats by integrating a Cloud Server with websockets.

Significance

Our project is significant in the realm of IoT and remote monitoring, utilizing advanced devices like ESP32CAM and ESP32-S3-DevKitC-1. The integration of cloud technology and websockets enables real-time monitoring, providing flexibility and convenience in remote interactions.

Cloud Server with WebSockets

A critical component, the Cloud Server with websockets facilitates real-time communication, allowing users to receive live video feeds and send control commands. Positive results showcase robust and reliable remote access and control, enhancing the system's flexibility for various monitoring applications.


Join us in exploring the potential of IoT in remote surveillance, offering intuitive features like live video streaming and image capture through our Web App SmartHome Security.
